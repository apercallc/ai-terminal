# Security Policy

## Supported Versions

Security fixes are provided for the latest released version.

## Reporting a Vulnerability

Please report security issues **privately**.

- Email: security@aiterminal.app
- Include: affected version, reproduction steps, impact, and any proof-of-concept.

We aim to respond within 72 hours and will coordinate a fix and disclosure timeline.

## Responsible Disclosure

- Please do not publicly disclose vulnerabilities before we confirm and patch.
- Please do not access or modify data that is not your own.

## Scope

In scope:
- The macOS desktop app (Tauri)
- The marketing/docs website in `docs/`
- Release artifacts and download pipeline

Out of scope:
- Third-party services and user-managed local LLM servers
