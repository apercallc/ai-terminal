# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-01

### Added

- Initial release of AI Terminal for macOS
- Full terminal emulator powered by xterm.js and portable-pty
- AI agent with goal-driven command planning and execution
- Support for OpenAI, Anthropic (Claude), and Local LLM providers
- Safe Mode with per-command approval workflow
- Auto-Accept Mode for trusted usage
- Command safety analysis with 5 risk levels and blacklist
- macOS Keychain integration for secure API key storage
- JSONL audit logging with daily rotation
- Dark and light theme support
- Command execution history with search and filtering
- Session metrics and analytics
- Settings panel with provider configuration and connection testing
- Configurable retries, timeouts, and scrollback limits
